Welcome to TeaScript's documentation!
=====================================

TeaScript is a powerful JavaScript golfing language created by StackExchange, PPCG, user `Downgoat <http://codegolf.stackexchange.com/users/40695/downgoat>`_. TeaScript compiles to JavaScript and contains many helpful features for golfing
